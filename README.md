# Proyecto Roadmap Java - POO

Este proyecto contiene ejemplos prácticos de los principales temas de Programación Orientada a Objetos (POO) en Java, desarrollados como parte del aprendizaje basado en el Roadmap de Java.

## Temas Implementados

- Modificadores de Acceso
- Paquetes
- Enumeraciones (Enums)
- Static Keyword
- Initializer Block
- Encapsulación
- Herencia
- Abstracción
- Interfaces
- Method Overriding
- Exception Handling

## Objetivo

Aplicar los conceptos fundamentales de Java mediante ejemplos prácticos que permitan comprender el funcionamiento de la Programación Orientada a Objetos.

## Tecnologías Utilizadas

- Java
- Eclipse IDE
- Página de Roadmap
